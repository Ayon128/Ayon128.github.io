## Essential Files

* `index.html` - Main homepage and first impression
  * Update hero section, social links, About Me, and Recent News

* `background.html` - Academic background and profile details
  * Education timeline and background narrative
  * Keep this aligned with current CV

* `publications.html` - Research publications list
  * Add/update papers, venue, status, and links (paper/GitHub)
  * Keep author highlighting and year-wise grouping consistent

* `achievements.html` - Awards and recognitions
  * Add competition results, honors, and notable milestones
  * Keep chronology and concise descriptions

* `projects.html` - Project portfolio
  * Showcase technical projects with summaries
  * Include repository/demo links where available

* `notes/index.html` - Notes listing page
  * Add/remove note cards and metadata (topic/date/read-time)
  * Controls which notes are visible in the Notes section

* `assets/css/style.css` - Global styles
  * Site-wide UI styling (cards, badges, nav, buttons, icons)
  * Update theme colors and component behavior consistently

* `assets/js/main.js` - Shared frontend behavior
  * Mobile menu, dark mode toggle, and page interactions
  * Keep script updates backward-compatible with all pages
